## Voie B locataire-first (validée)

**Audience principale** : locataires FR cherchant à vérifier leur loyer / DPE / dossier / défense en cas de litige.

**Pourquoi Voie B vs Voie A bailleur** :
- Volume audience B2C locataire >> volume bailleur (~30M vs ~6M en FR)
- DossierFacile (gov.fr) crée demande Voie A déjà saturée
- BailleurVérif a moat propre Voie B (observatoire conformité indépendant + RAG jurisprudence + générateur courriers)
