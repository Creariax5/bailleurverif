Widget DPE gratuit — embed 1 ligne

Badge JavaScript pour afficher le statut DPE d'un logement (conforme / interdit location) sur votre blog immo, votre site d'agence, ou votre comparateur. Gratuit · sans iframe · sans tracker tiers · 0 dépendance.

Code à copier-coller

Carte standard (380px max-width)

<script src="https://bailleurverif.fr/widget/dpe-status.js"
        data-ville="paris"
        data-dpe="F"

<script src="https://bailleurverif.fr/widget/dpe-status.js"
        data-ville="paris"
        data-dpe="F"

        async></script>

<script src="https://bailleurverif.fr/widget/dpe-status.js"

Pourquoi ce widget ?

  • Crédibilité immédiate : DPE F/G = sujet juridique brûlant, source Loi Climat & Résilience citée.
  • 0 iframe : DOM léger, hérite des couleurs parent si vous voulez, indexable.
  • 0 tracker tiers : un seul appel pixel statistique anonyme (ville+classe DPE), pas de cookie, pas de fingerprint. RGPD-friendly.
  • 0 dépendance : pas de jQuery, pas de React, pas de framework. ~2,5 KB gzip.
  • Bidirectionnel : vous embeddez → vos lecteurs cliquent → ils vérifient gratuitement leur conformité bailleur sur BailleurVérif.

Cas d'usage typiques

  • Blog immobilier : article "Investir à Lille en 2026" → afficher le DPE moyen de la ville + interdiction location associée.
  • Agence immo : fiche bien individuelle → badge "DPE F · Interdit nouvelle location 2028" pour transparence vendeur/acquéreur.
  • Comparateur location/achat : afficher la classe DPE et son impact réglementaire à côté du loyer.
  • Calculateur rentabilité locative : avertir avant achat sur l'interdiction location à terme.

RGPD & vie privée

Le widget enregistre un compteur de vue anonyme (ville + classe DPE + horodatage). Pas d'ID utilisateur, pas de cookie, pas de fingerprint navigateur, pas de tiers ad-tech. Aucune donnée personnelle traitée. Vous pouvez l'utiliser sans bandeau consentement.
