Comment l'installer

WordPress

Ajoutez un bloc HTML personnalisé et collez le snippet. C'est tout. Fonctionne avec Gutenberg, Elementor, Divi, Bricks.

Webflow

Glissez un élément Embed sur la page, collez le snippet, publiez.

Shopify · Squarespace · Wix

Ajoutez un bloc Code / Embed HTML et collez le snippet.

Site HTML statique (Jekyll, Hugo, etc.)

Collez le snippet directement dans le markdown ou le template HTML, à l'endroit voulu.

Questions fréquentes

Le widget est-il vraiment gratuit ?
Oui, 100% gratuit, sans limite d'impressions, sans inscription, sans clé d'API. Aucune intention de monétiser le widget côté éditeur.
Quelles données sont mises à jour automatiquement ?
Les barèmes encadrement de loyer 2026 des 31 communes en zone tendue (Paris, MEL, Métropole Lyon, Bordeaux, Montpellier, Plaine Commune, Est-Ensemble, Grenoble-Alpes Métropole), le calendrier DPE (G interdit 2025, F 2028, E 2034), les aides MaPrimeRénov' 2026 et le déficit foncier majoré 21 400 €/an prolongé jusqu'en 2027.
Le widget est-il RGPD-compliant ?
Oui. Aucun cookie déposé. Aucun tracker tiers chargé. Une requête d'impression anonymisée (outil + ville + referer + timestamp) est envoyée à BailleurVérif pour suivi statistique global, sans IP ni identifiant utilisateur. Vous n'avez aucune bannière supplémentaire à afficher.
Puis-je personnaliser l'apparence ?
La taille (largeur × hauteur) est libre via les attributs width/height. La taille recommandée est 320×440. Pour aller plus loin, contactez-nous via le bouton « Donner un feedback » du site principal.
Comment savoir si des visiteurs cliquent sur le widget depuis mon site ?
Les liens sortants contiennent automatiquement des paramètres UTM (utm_source=embed, utm_medium=widget, utm_campaign=<tool>). Côté votre site, ajoutez un événement clic via votre tag manager (Google Tag Manager, Plausible, Matomo) sur l'iframe pour mesurer l'engagement.
Y a-t-il une limite de domaines / impressions ?
Aucune limite. Vous pouvez l'intégrer sur autant de pages / sites que souhaité.
Qui se cache derrière BailleurVérif ?
BailleurVérif est un outil indépendant qui agrège les règles bailleur 2026 (encadrement, DPE, anti-fraude dossier) en un diagnostic instantané et gratuit. Voir l'outil complet →
