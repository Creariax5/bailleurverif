Pourquoi ce flux ?

  • ★ Loi Climat & Résilience : les logements classés G sont interdits à la location depuis le 1er janvier 2025. Les F le seront au 1er janvier 2028.
  • ★ Un diagnostic F/G publié aujourd'hui est souvent le signe d'un logement sur le point d'être remis sur le marché (vente / re-location). Le seul moyen pour un locataire de l'identifier en amont, c'est ce flux.
  • ★ Aucun signal n'existait jusqu'ici : l'ADEME publie en lots, sans agrégation par commune ni format consultable.
  • ★ Les bailleurs concernés peuvent calculer leurs aides à la rénovation ou utiliser notre simulateur déficit foncier.

Sources et méthode

  • • Données source : ADEME — DPE Logements existants (depuis juillet 2021), ~14.7 millions de diagnostics indexés. Licence Ouverte 2.0 Etalab.
  • • Méthode : requête API filtrée par etiquette_dpe IN (F, G) et code_insee_ban pour chacune des 54 zones, tri descendant par date_etablissement_dpe, dédoublonnage par hash SHA1 (date+INSEE+rue+numéro+surface+étiquette).
  • • Périodicité : crawl automatique toutes les 6 heures (cron VPS), 25 records max par commune et par crawl.
  • • Vie privée : l'ADEME publie ces données en open data avec adresse complète. Cette page ré-affiche uniquement la donnée publique. Aucune information sur le propriétaire / occupant n'est diffusée.
  • • API : GET /api/observatoire-dpe-fg?insee=75101&limit=50 (réponse JSON, aucune authentification).
