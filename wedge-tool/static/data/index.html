Source & méthodologie

Arrêtés préfectoraux 2026 publiés par les préfectures concernées (Paris, Nord, Rhône, Gironde, Hérault, Seine-Saint-Denis, Isère) conformément aux articles L.302-7-1 et suivants du Code de la construction et de l'habitation. Les plafonds représentent un loyer de référence (médian + 20%) — le plafond exact applicable à un logement spécifique dépend du secteur OLAP précis, du nombre de pièces, du caractère meublé/nu et de l'époque de construction. Le calculateur interactif BailleurVérif couvre ces nuances commune par commune.

Usage code

Python (pandas) :

import pandas as pd
df = pd.read_csv("https://bailleurverif.fr/data/encadrement-loyer-france-2026.csv")
print(df[["commune", "plafond_nu_eur_m2"]].sort_values("plafond_nu_eur_m2", ascending=False))

JavaScript (fetch) :

const r = await fetch("https://bailleurverif.fr/data/encadrement-loyer-france-2026.json");
const { communes } = await r.json();
console.log(communes.find(c => c.slug === "paris"));

Formats avancés & métadonnées

Pour faciliter l'intégration dans des outils tiers (chaînes ETL, plateformes open-data, gestionnaires de citations) :

  • Méthodologie détaillée — sources par préfecture, définition du plafond, audit trail, limites connues.
  • datapackage.json — descripteur Frictionless Data Package v1 avec schéma typé complet (types, contraintes, primaryKey).
  • CITATION.cff — Citation File Format 1.2.0 (GitHub-natif, gestionnaires de citations académiques).
  • catalog.xml — catalogue DCAT-AP 2.1 (RDF/XML, format pivot data.europa.eu).
