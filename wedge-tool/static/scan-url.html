Vérifiez une annonce en 5 secondes

Collez l'URL d'une annonce Locservice.fr : on extrait automatiquement le loyer, la surface, le DPE et la commune, puis on vérifie la conformité (encadrement préfectoral 2026 + Loi Climat DPE). Verdict PNG share-friendly à télécharger pour Twitter, Reddit, WhatsApp. Anonyme, sans inscription, sans cookie.

PAP, SeLoger, Leboncoin : prévus v1 (pages JS-rendered hostiles).

(studio 16 m² Paris 15ᵉ à 1 195 €/mois — 86 % au-dessus du plafond légal, observatoire BV)

Comment ça marche

  1. Vous collez l'URL d'une annonce Locservice publiée publiquement.
  2. Notre serveur fait une seule requête HTTP polie (UA identifié, pace 30s) pour récupérer la page.
  3. On extrait le loyer affiché, la surface (depuis JSON-LD Apartment structuré), le DPE (image filename energie-X.png) et le code postal.
  4. On compare aux plafonds d'encadrement préfectoral 2026 (31 communes couvertes) et au calendrier DPE Loi Climat (G interdit depuis 2025-01-01).
  5. Verdict affiché immédiatement, PNG verdict téléchargeable pour partage.

Pourquoi ces sources

Plafonds d'encadrement = arrêté préfectoral 2026 par commune (Paris, Lyon, Bordeaux, Lille, Montpellier, Grenoble, EPT 93 — 31 communes au total). Calendrier DPE = loi Climat & Résilience 2021 : G interdit pour les nouveaux baux depuis le 1er janvier 2025, F dès 2028, E dès 2034. Notre observatoire d'annonces non-conformes (232 annonces ouvertes, dataset Etalab v2.0) sert de baseline d'audit.

Limites du V0

  • Locservice uniquement (robots.txt permissif). PAP, SeLoger, Leboncoin : JS-rendered + Cloudflare → V1 prévu via headless.
  • Plafond d'encadrement = valeur maximale par commune (V0). Le vrai loyer médian majoré dépend de l'arrondissement × type × époque. Si nous flagons "presumed", c'est probable mais pas définitif — vous gagnez une limite haute.
  • Notre lecture du DPE = image filename. Pas d'OCR. Si Locservice ne publie pas l'image, on renvoie un verdict partiel.
  • Aucune donnée personnelle vendeur stockée : seul le hash de l'URL est tracé pour statistiques anonymes.
