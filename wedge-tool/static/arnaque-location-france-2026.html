    Les 8 drapeaux rouges qui s'appliquent partout en France

    1. Paiement Western Union / MoneyGram / Bitcoin / PCS / Neosurf — aucun bailleur sérieux ne demande ces moyens. Drapeau rouge maximal, vrai partout (Paris comme Strasbourg).
    2. Bailleur déclaré à l'étranger ou en mission — "je suis à Londres / Dubai / Casablanca, envoyez par chronopost" : pattern classique. Évidemment, certains bailleurs SONT expatriés ; mais la combinaison "absent" + "payez d'abord" est éliminatoire.
    3. Demande de caution ou loyer avant la visite physique — illégal en France (loi du 6 juillet 1989). Ne payez rien sans avoir visité.
    4. Prix anormalement bas — sous ~5€/m² dans une métropole, c'est statistiquement impossible. Appât classique pour engager. Les fiches locales ci-dessus donnent un repère par ville.
    5. Basculement WhatsApp / Telegram / SMS dès le 1ᵉʳ message — sortir de la plateforme = échapper à sa modération. Refusez et insistez pour rester sur le canal officiel jusqu'au RDV de visite.
    6. Ton d'urgence injustifié — "URGENT", "libre tout de suite", "plusieurs candidats", "réservez sous 24h". Pression psychologique = signal fraude.
    7. Photos envoyées par email/WhatsApp au lieu d'être dans l'annonce — souvent volées sur d'autres annonces ailleurs. Reverse-image-search Google les retrouve.
    8. Adresse précise absente — un bailleur sérieux mentionne au moins la rue ou le métro. Adresse floue = suspect.

    Que faire si vous avez déjà versé de l'argent

    1. Appelez immédiatement votre banque : un virement SEPA peut parfois être rappelé sous 13 mois pour fraude avérée.
    2. Déposez plainte au commissariat de votre ville (pré-plainte en ligne possible sur pre-plainte-en-ligne.gouv.fr).
    3. Signalez sur cybermalveillance.gouv.fr : ils proposent un parcours assisté + statistiques officielles.
    4. Conservez TOUS les échanges : captures écran de l'annonce + texte intégral + emails + SMS + accusés virement + RIB du fraudeur. Indispensable pour les 3 démarches ci-dessus.
    5. Signalez l'annonce sur la plateforme (Leboncoin, SeLoger, PAP ont tous un bouton signalement) pour éviter qu'elle continue à piéger.

    Sources et méthodologie

    Les estimations cas/mois par ville sont des extrapolations BailleurVérif fondées sur (a) les statistiques agrégées cybermalveillance.gouv.fr pour la catégorie "fraude au logement", (b) les remontées plateformes (Leboncoin, SeLoger, PAP) lorsqu'elles publient des bilans, (c) une pondération par population/densité étudiante locale (proxy d'exposition). Ces chiffres sont des ordres de grandeur, pas des mesures directes ; la sous-déclaration est probable.

    Le scanner anti-arnaque utilise une heuristique sur le texte (regex sur les 8 patterns documentés ci-dessus) et retourne un score 0–100 + verdict. Le texte n'est pas stocké en base. Le code est public sur GitHub MIT — pull requests bienvenus pour améliorer les heuristiques.
