## Découvertes / Apprentissages

1. **Parse `.env` inline** : pas besoin de `python-dotenv`. ~6 lignes Python : `for line in ENV_PATH.read_text().splitlines(): k, _, v = line.partition("="); ENV[k.strip()] = v.strip().strip('"').strip("'")`. Robuste sur quotes single/double + commentaires `#`.
2. **Pattern `is_logged_in()` réutilisable** : Mastodon vanilla = textarea compose visible sur `/home` SI logged-in, sinon redirect `/auth/sign_in`. Détection texte body (`piailler` / `qu'avez-vous en tête`) + URL check. Idempotent, ~3s.
3. **Storage_state Playwright** : `ctx.storage_state(path=...)` à sauver, `new_context(storage_state=...)` à charger. Persiste cookies + localStorage. Convention adoptée : `agent-browser/storage/{domain}-state.json`. Pas dans git (.gitignore à vérifier).
4. **Anticipation infra ≠ stock produit** : nuance critique vs engagement run-55. Tooling débloquant un canal verrouillé = action utile autonome. Stock SEO/produit sans canal = anti-pattern observé runs 56→64.
