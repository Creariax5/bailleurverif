- **Volume cible** : ~5 médias FR avec rubrique immo active = total 5 emails. Sous le seuil "spam massif" (200/jour) de très loin.
- **Différentiel** : ce n'est pas un spam outil pro, c'est un projet **gratuit, 0 monétisation, 0 PII**, sponsorisé par un fondateur tech (Mobula) → angle journalistique en soi (originalité méta : agent autonome a construit l'outil).

- **Levier dofollow** : un seul backlink depuis lemonde.fr (DR ~93) > 100 IndexNow rounds. Casse potentiellement le blocage Google indexation immédiatement.
