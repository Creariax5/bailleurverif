Allowlist channels : whatsapp, email, copy, sms, telegram, messenger, linkedin, twitter. Ajout dans `/api/stats` : `shares_total`, `shares_unique_sessions`, `share_rate_pct`, `share_channels`, `referrals_from_share` (compte les `referrer` contenant `src=share`).

- `wedge-tool/static/app.js` : fonctions `share(channel)` et `fallbackCopy(text)`. Le texte partagé est **adapté à la sévérité du verdict de l'utilisateur** :
  - severity `danger` : "Je viens de vérifier en 30 sec si ma loc est conforme à la loi 2026 — gros pain découvert..."
  - severity `warn` : "J'ai testé un outil qui vérifie en 30 sec..."
  - severity `ok` : message générique
  - Si la ville est connue ET severity != ok, on personnalise "ma loc à {Ville}"
  - URL partagée : `http://217.182.171.135:8102/?src=share&via={channel}` → permet de mesurer `referrals_from_share` côté stats
- `dashboard/metrics_sync.sh` : ajout des 5 nouveaux champs partage dans le bloc `metrics["wedge"]`.
- `dashboard/live.html` : nouvelle ligne dans le bloc wedge "Partages : N total · X% taux · arrivées via partage : N · canaux : ...".

**Tests** :

- POST `/api/share` valide ({whatsapp, ok, paris}) → `{"ok":true}`. POST canal invalide → `{"ok":false,"error":"invalid channel"}`. Stats incrémentées.
- Cleanup smoke test : `s-smoke-test` retiré de `shares.jsonl` (fichier vide). Vrai compteur intact.

**Pourquoi ce design** :
- **Position après les email-gates** : protège la conversion email (= go/no-go). Le partage est secondaire.
- **Texte adapté au verdict** : un utilisateur en `danger` est légitime à crier ("gros pain"), un utilisateur `ok` ne doit pas mentir → pas de fake. C'est ce qui maximise le partage authentique.
- **Channel `copy`** : indispensable pour qui veut le coller sur un canal non couvert (Slack, Discord, forum, etc.) — multiplicateur sans rien construire.
- **Pas de tracker tiers** : 0 GDPR, 0 cookies. Tracking interne JSONL.
- **K-factor potentiel** : si 10% des visiteurs partagent et chaque partage = 0.5 nouveau visiteur, K = 0.05. Faible mais > 0, et c'est la SEULE source autonome aujourd'hui.
