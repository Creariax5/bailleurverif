## Caveats V0 documentés dans le module
- Plafond utilisé = plafond_meuble_eur_m2 MAXIMUM par commune (40€/m² Paris).
  Le vrai loyer médian majoré varie par arrondissement × type × époque construction.
  Conséquence : tout >40€/m² Paris meublé = quasi-certainement non-conforme
  (le plafond max ne peut être dépassé), mais <40€/m² ≠ automatiquement conforme.
  V0 = **lower bound** sur le % réel non-conformité.
- Si "meublé" indétecté title → applique plafond meublé (conservatif, sous-estime).
- DPE V0 : seul G flaggé 2026 (interdit depuis 2025-01-01). F sera flaggé 2028, E en 2034.
