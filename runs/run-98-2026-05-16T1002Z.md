## ASSESS

### Pourquoi cette seconde salve en priorité

- **Élargissement géographique massif** : encadrement loyer = 31 communes zones tendues. DPE = **toute la France** (10M propriétaires bailleurs, ~5,2M passoires thermiques nationales).
- **Intent transactionnel élevé** : un bailleur qui cherche « dpe f interdit location {ville} » a une décision à prendre dans les 6-24 mois (vendre, rénover, perdre le revenu locatif). Conversion landing → outil principal élevée.
- **Concurrence SEO faible-modérée** sur longtail `{ville} dpe f` (essentiellement service-public.fr générique + Pap/SeLoger sans simulateur par ville). Différenciation : **estimation locale du parc de passoires + mini-simulateur instantané "ma classe = quand interdite ?"**.
- **0 dépendance externe** : tout client JS, données réglementaires en dur citant loi 2021-1104 + article 17-21-bis + décret 2022-1726.
- **0 dépense**.

### Risques évalués + mitigations

- **Données chiffrées par ville = sensible** (afficher un mauvais chiffre = perte crédibilité). Mitigation :
  - Ratios national 17% (ONRE/ADEME 2024) explicitement cité comme source.
  - Ajustement local 12-22% selon âge moyen du parc (Paris/Marseille/Lyon centre = ratio supérieur ; villes neuves = inférieur).
  - **Footer disclaimer** : « Les chiffres présentés sont des estimations basées sur les ratios nationaux ONRE/ADEME 2024 appliqués au parc local. Pour le diagnostic exact, consultez l'observatoire ADEME des DPE. »
  - Nombres affichés en « ≈ » (approximation explicite).
- **Calendrier juridique évolutif** : un report par décret (cf. proposition LR/RN 2025) pourrait obsolète le contenu. Mitigation : citations légales précises (article + décret), refresh build trivial si décalage.
- **JSON-LD trop lourd ou conflit** : test parse OK sur 3 échantillons (6 nœuds @graph).
