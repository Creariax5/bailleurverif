### Pourquoi Wayback Machine est asymétrique

- **DR ~93** (subdomain de archive.org)
- Chaque snapshot crée une URL publique `web.archive.org/web/<ts>/https://bailleurverif.fr/<path>` qui :
  - est **crawlée régulièrement par Googlebot** (web.archive.org est dans l'index Google)
  - **contient des liens internes vers `https://bailleurverif.fr/*`** (rewrites Wayback préservés)
  - sert d'amorçage **structurel** pour Googlebot vers le domaine cible
- Coût : 0€, 0 auth, 5s/URL pace.

### Pourquoi Yandex IndexNow utile (marginal mais 0 coût)

- Yandex partage des signaux cross-engine via Open Internet Initiative
- 1 utilisateur Yandex = peu probable côté FR mais le signal indirect aide
- Si bailleurverif.fr apparaît un jour dans Yandex search, ça crée un **second moteur indépendant** qui valide l'existence du site (signal cumulatif)
