## ASSESS

### Pourquoi ce tool en priorité

- **Diversification public** : encadrement loyer = niche bailleurs en zone tendue. Préavis bail = grand public locataires + bailleurs partout en France.
- **Intent transactionnel haut** : quelqu'un qui cherche son préavis a une décision concrète à prendre dans les 24-48h → conversion élevée vers l'outil principal.
- **Couverture règle simple, deliverable concret** : durée + date butoir + modèle LRAR copier-coller. ROI utilisateur immédiat.
- **0 dépendance externe** : pas d'API tierce, calcul 100% client JS, données réglementaires en dur citant loi 1989 art. 15 + art. 25-8 + Alur 2014 + ELAN 2018.
- **Aucune dépense**.

### Risques évalués + mitigations

- **Concurrence SEO élevée** sur `préavis bail` (existing pages service-public.fr, Pap, SeLoger). Mitigation : différenciation par **simulateur interactif + modèle LRAR pré-rempli** (rares concurrents font les deux). Longtails `préavis bail meublé zone tendue` moins concurrencés.
- **Information juridique erronée** = catastrophique. Mitigation : verdict explicitement informatif (footer + résultat), références loi citées, renvoi ADIL pour litige.
- **Calcul date butoir** : edge cases (29/02, mois 31→30 jours). Mitigation : `Date.setUTCMonth(month + N)` JS arithmétique native, comportement standard (rollover automatique). Vérifié mentalement sur 4 cas.
